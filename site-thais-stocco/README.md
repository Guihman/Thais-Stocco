# Site Thais Stocco | Psicóloga

Site responsivo de página única, pensado para ser usado na bio do Instagram.

## Como editar rapidamente

Abra o arquivo `index.html` e troque:

1. `https://wa.me/5500000000000` pelo WhatsApp real.
   - Exemplo: `https://wa.me/5511999999999`

2. `CRP 00/000000` pelo CRP real.

3. `Rua Exemplo, 123 — Cidade/UF` pelo endereço real ou região.

4. `thaisstocco@email.com` pelo e-mail real.

5. `https://instagram.com/` pelo Instagram real.

6. O bloco `Foto da Thais` pode ser substituído por uma foto.
   - Coloque a foto na pasta do site, por exemplo `assets/thais.jpg`.
   - Depois altere no HTML/CSS conforme necessário.

## Como publicar

Opções simples:
- Netlify: arraste a pasta do site para o painel do Netlify.
- Vercel: importe a pasta/projeto.
- GitHub Pages: envie os arquivos para um repositório e ative Pages.

Arquivos:
- `index.html`: estrutura e textos.
- `styles.css`: design e animações visuais.
- `script.js`: menu, animações, partículas e interações.
